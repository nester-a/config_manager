# Задание 1: Менеджер конфигурации (Пакеты, Модули, Коллекции)

**Цель:** Закрепить структурирование кода и работу с `HashMap`.

**Задача:**
Создайте пакет `config_manager` со следующей структурой модулей:
* `src/lib.rs` - корень библиотеки
* `src/config.rs` - модуль для структуры конфигурации
* `src/loader.rs` - модуль для загрузки конфигурации

**Требования:**

1. В `config.rs` определите структуру `AppConfig` с полями:
* `database_url: String`
* `port: u16`
* `feature_flags: HashMap<String, bool>`

2. В `loader.rs` создайте функцию `load_config() -> Result<AppConfig, String>`, которая:
* Создаёт и возвращает экземпляр `AppConfig` с заранее заданными значениями
* Используйте `HashMap` для фич-флагов (например, `("dark_mode", true)`)

3. В `lib.rs` используйте модули и переэкспортируйте основные типы/функции.

**Подсказка:** Не забудьте добавить необходимые `use-` строки и объявления модулей.
